# 02_heatmap_features_harmonization

This folder contains the `Heatmap2_*` scripts copied from the original `codes/` directory.

These scripts cover heatmap-derived feature generation, ROI/network summaries, ComBat harmonization where applicable, harmonized outputs, plots, group comparisons, regressions, and partial-correlation analyses. The folder name keeps harmonization visible as an important processing label, but the stage is broader than harmonization alone.

This stage is downstream of gradient/heatmap generation in `code/01_gradient_heatmap_generation/` and upstream of mSFPCA trajectory analyses in `code/03_mfpca_trajectories/`.

The scripts retain the original analysis logic and may require local path edits before use on approved study data.
