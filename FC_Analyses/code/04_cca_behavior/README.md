# 04_cca_behavior

This folder contains `CCA4_*` scripts and `UNC_Tra_Desc_REF.m` copied from the original `codes/` directory.

These scripts construct brain and behavioral matrices from trajectory-derived measures, configure CCA/PLS Toolkit analyses, run descriptive and predictive CCA/PLS models, and summarize latent variable results. This stage follows mSFPCA trajectory modeling in the reviewer-facing workflow.

The scripts retain the original analysis logic and may require local path edits plus CCA/PLS Toolkit setup before use on approved study data.
