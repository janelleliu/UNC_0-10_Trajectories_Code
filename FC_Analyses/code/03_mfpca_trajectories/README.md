# 03_mfpca_trajectories

This folder contains `MSFPCA3_*` scripts copied from the original `codes/` directory.

These scripts construct longitudinal feature tables, estimate/reconstruct developmental trajectories, and run trajectory-level statistical analyses. This stage follows gradient/heatmap generation plus heatmap-feature/harmonization analyses, and it provides trajectory-derived features for downstream CCA and BAG-related analyses.

The scripts retain the original analysis logic and may require local path edits before use on approved study data.
