# Simulated demo data

These files are synthetic and contain no real participant data. They are provided only to demonstrate the CCA-style input format and demo execution.

- `simulated_subjects.csv`: subject IDs and simple grouping variables.
- `simulated_brain_features.csv`: simulated brain-derived variables resembling network, gradient, or trajectory summary features.
- `simulated_behavior.csv`: simulated behavioral variables.
- `simulated_covariates.csv`: simulated covariates used for optional residualization before CCA.

The simulation includes shared latent structure between brain and behavior variables so that the CCA demo produces interpretable canonical correlations. The data are not intended to reproduce any manuscript result.
